# 1. Project overview
This project demonstrates the deployment of a secure static website on AWS using Amazon EC2, Ubuntu, and Nginx. The infrastructure includes a custom VPC, public subnet, Internet Gateway, Route Table, Security Groups, VPC Flow Logs, and a custom domain configured with DNS. The website is secured with HTTPS using Let's Encrypt SSL certificates, providing encrypted communication and a production-style hosting environment.

Through this project, I gained hands-on experience with AWS networking, Linux server administration, DNS configuration, web server deployment, and cloud security best practices.

## Objective

- Deploy a static website on an Amazon EC2 instance using Ubuntu and Nginx.
- Build a secure AWS networking environment with a custom VPC, public subnet, Internet Gateway, Route Table, and Security Group.
- Configure a custom domain to point to the EC2 instance using DNS.
- Secure the website with HTTPS using Let's Encrypt and Certbot.
- Enable VPC Flow Logs for network monitoring and security visibility.
- Gain hands-on experience with AWS infrastructure, Linux administration, web hosting, and cloud security best practices.

## Deliverable
A publicly accessible landing page served by Nginx on an Ubuntu EC2 instance inside a purpose-built AWS network.
